# 地滑りマップ（KMZ/KML） — Google Earth Pro 用

  このリポジトリは、デスクトップ版 Google Earth Pro で閲覧する地滑り関連の地理空間データ（KMZ/KML）を公開します。  
  **対応OS:** Windows / macOS / Linux（モバイル版 Google Earth は対象外）

  ## 使い方
  - Google Earth Pro（公式）: <https://www.google.com/earth/about/versions/#download-pro>
  - 本リポジトリの KMZ/KML をダウンロードし、ダブルクリックで開くか **[ファイル] → [開く]** から読み込みます。  
    インストール手順や要件は上記公式ページをご参照ください。

  ## データ内容
  - **a#_dfm_roi** — だいち（ALOS）の **北行軌道** 観測に基づき、地滑りと考えられる地形的特徴が見られた地域  
    - `#=01`: 2019-11-22 と 2021-11-19 の観測  
    - `#=02`: 2019-11-22 と 2022-12-02 の観測  
    - `#=03`: 2021-11-19 と 2022-12-02 の観測
  - **b#_dfm_roi** — 上記と同様だが **南行軌道** 観測に基づく  
    - `#=01`: 2019-11-05 と 2021-08-10 の観測  
    - `#=02`: 2019-11-05 と 2023-02-21 の観測  
    - `#=03`: 2021-08-10 と 2023-02-21 の観測
  - **地すべりマップ** — 防災科学技術研究所（NIED）による地すべり痕跡の判読結果  
  - **ロッジの位置.kmz** — 椹島ロッジ、二軒小屋ロッジ、千枚小屋の位置

  > 参考: J-SHIS 地すべりマップ <https://www.j-shis.bosai.go.jp/landslidemap>
